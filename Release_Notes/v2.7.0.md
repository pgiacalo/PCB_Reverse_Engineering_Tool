# Release Notes - v2.7.0

**Release Date**: TBD

## Overview
Version 2.7.0 introduces BOM export functionality, Settings menu improvements, UI enhancements, and license additions.

## Major New Features

### BOM Export
- **Export BOM Feature**: Added Export BOM feature to File menu
- Export Bill of Materials for all components in the project
- Useful for documentation and procurement

### Change Perspective Dialog
- **Eye Icon Button**: Added eye icon button to toolbar for quick access to Change Perspective dialog
- Simplified Change Perspective dialog view switching
- Improved perspective change workflow

## UI/UX Improvements

### Camera and View System
- **World Coordinates**: Refactored camera view to use world coordinates for consistent perspective changes
- Fixed Set View command to use world coordinates for camera view
- Improved perspective transformation accuracy

### Image View Switching
- Fixed bottom image position when switching to/from bottom view
- Fixed Power/Ground icon rotation/flipping during image view switching

## Technical Improvements

### License and Attribution
- **MIT License**: Added MIT License headers to all code files
- Created LICENSE file
- Added explicit Author attribution to all files
- Removed redundant Author lines from license headers

### Code Quality
- Fixed TypeScript build errors
- Improved code organization and consistency
