# Release Notes - v2.4.0

**Release Date**: TBD

## Overview
Version 2.4.0 introduces improvements to image management, UI enhancements, and better project setup workflow.

## UI/UX Improvements

### Image Management
- **Brightness and Contrast Controls**: Added brightness and contrast controls to Image Management section
- **Automated Transparency Slider**: Added automated transparency slider functionality
- **Edge Detection Removal**: Removed Edge Detection mode from Image Management section

### About Menu Updates
- Updated About menu to emphasize reverse engineering and troubleshooting capabilities
- Updated second paragraph to begin with "This tool can assist in troubleshooting by tracing..."
- Added '14 independent layers to manage PCB element types' as first item in Layer Management section
- Removed Erase tool from Tools and Shortcuts sections

### Project Setup
- **Simplified New Project Setup**: Request readwrite permission upfront to reduce new project setup to single dialog

## Bug Fixes

### Power/Ground Tools
- Fixed power and ground bus icon colors to match assigned bus colors
- Fixed Power/Ground tool selection issues

### Labeling
- Removed '(Signal)' from via labels for cleaner display

## Version Information
- **Version**: 2.4.0
- **Previous Version**: 2.3.0

## Migration Notes
- No breaking changes
- All project files from previous versions remain fully compatible
- No migration required
