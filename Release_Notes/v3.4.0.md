# Release Notes - v3.4.0

## New Features
- **Zig-Zag Pattern Placement**: Added a new arrangement type for automated via and pad placement. This pattern arranges pins in two rows or columns with a zig-zag numbering sequence, common in many connector and IC layouts.
- **Orientation Control for Zig-Zag Patterns**: Users can choose between Vertical (2 columns) and Horizontal (2 rows) orientations for the zig-zag arrangement.
- **Rotate 45Â° in Change Perspective**: Added a new rotation option to the Change Perspective dialog to rotate all elements by 45 degrees.

## Improvements
- **Enhanced Pattern Placement Dialog**: Updated the UI with icons and better validation for all arrangement types (Linear, 2-Sided, 4-Sided, and Zig-Zag).
- **Requirements Documentation**: Updated technical requirements to include pattern placement specifications.
- **README Updates**: Added automated pin/pad arrangement to the feature list.
