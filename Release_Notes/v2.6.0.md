# Release Notes - v2.6.0

**Release Date**: TBD

## Overview
Version 2.6.0 introduces major improvements including Find functionality, Help menu, auto-save enhancements, and various bug fixes.

## Major New Features

### Find Functionality
- **Find Buttons**: Added Find buttons to all object types in Information dialog
- Find buttons center the view on selected objects and bring them into focus

### Help Menu
- **Step-by-Step Usage Guide**: Added comprehensive Help menu with step-by-step usage guide
- Provides detailed instructions for using all tool features

### Auto-Save Improvements
- **Three-State Save Status Indicator**: Added visual indicator showing save status (saved, saving, unsaved)
- **Component Connections Feature**: Added "All Component Connections" feature to view all component connections
- Fixed auto-save to trigger properly when components are added

### PastTime (Restore from History)
- Added PastTime feature to restore project state from history

## UI/UX Improvements

### Tool Shortcuts
- Changed Zoom tool shortcut from Z to M
- Improved keyboard shortcut organization

### Image Management
- **Lowercase Images Directory**: Fixed image loading to use lowercase 'images' directory
- **Nested Directory Access**: Improved proper nested directory access for images
- **Image Storage Optimization**: Store images in Images/ subdirectory to reduce JSON file size

### Display Improvements
- Cleaned up display text in Information dialog
- Improved visual clarity throughout the application

## Bug Fixes

### View and Perspective
- Fixed Bottom View offset issue for components and drawing strokes
- Fixed rotation alignment issues
- Updated menu labels for better clarity

### Tool Cursors
- Fixed tool cursor icons to scale properly with zoom/magnify level

### Project Management
- Fixed project opening race condition
- Improved project file handling reliability

## Version Information
- **Version**: 2.6.0
- **Previous Version**: 2.5

## Migration Notes
- No breaking changes
- All project files from previous versions remain fully compatible
- No migration required
