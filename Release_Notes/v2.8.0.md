# Release Notes - v2.8.0

**Release Date**: TBD

## Overview
Version 2.8.0 focuses on perspective improvements, component indicator fixes, color palette reorganization, and keyboard shortcut enhancements.

## Major New Features

### Perspective Improvements
- **Component Indicators**: Component indicators now remain upright during perspective changes
- Fixed perspective switching and Set View feature
- Improved coordinate transformation for perspective changes

### Keyboard Shortcuts
- **Arrow Key Shortcuts**: Added left/right arrow key shortcuts for switching perspective
- Added up/down arrow key shortcuts for perspective rotation
- Updated Help/About menu styling with revised arrow key shortcuts

### Color Palette
- **Reorganized Color Palette**: Reorganized by color family with light-to-dark ordering
- Improved visual organization and color selection workflow

## UI/UX Improvements

### Hand Tool
- Fixed Hand tool to move objects instead of camera, with proper coordinate transformation
- Improved object manipulation accuracy

### Project Management
- **Dirty Indicator**: Track all project changes for dirty indicator and auto-save
- Improved project state tracking

### Splash Screen
- Updated splash text and alignment
- Improved visual presentation

## Technical Improvements

### Code Quality
- Consolidated math operations for better performance
- Fixed TypeScript unused variable errors
- Improved code organization
